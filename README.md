# octopackage

> [octorelease](https://github.com/mizzy/octorelease) for Composer

## Install

```console
$ composer require k1low/octopackage --dev
```

## Usage

```console
$ cd your-php-project/
$ composer octopackage
```

### `--major` `--minor` `--patch`

Version number when package. default is `--patch`
